MANUAL DE SUPERVIVENCIA WEB

GUÍA TÉCNICA PARA DISEÑADORES Y PROGRAMADORES

01. ESTRUCTURA (HTML)

El esqueleto de toda página. Usa etiquetas con significado para que los buscadores te encuentren.

  • <main> : Contenido principal.
  • <section> : Bloques de contenido.
  • <aside> : Barras laterales o extras.
  • <article> : Contenido independiente.

02. ESTILO (CSS)

Atajos visuales para mantener un diseño limpio y moderno.

/* CENTRADO ABSOLUTO */
display: flex;
justify-content: center;
align-items: center;
  • Flexbox: Ideal para filas y columnas simples.
  • Grid: Para estructuras complejas de revistas.
  • Variables: --color: #fff; para reutilizar tonos.

03. ATAJOS DE TECLADO (VS CODE)

Gana velocidad en el desarrollo con estos comandos esenciales.

CMD/CTRL + DSeleccionar misma palabra
ALT + ↑ / ↓Mover línea de código
CMD/CTRL + /Comentar bloque
ALT + CLICKMúltiples cursores

04. LÓGICA (JAVASCRIPT)

Acciones rápidas para manipular tu web sin complicaciones.

  • document.querySelector() : Atrapa cualquier cosa.
  • addEventListener('click', ...) : Detecta toques.
  • classList.toggle() : Prende y apaga clases CSS.
